<h2>Face Detection using OpenCV</h2>
  
## Description
This project implements a real-time face detection system using Python and OpenCV. It uses a pre-trained Haar Cascade Classifier to detect human faces from a webcam video stream or images. The system processes each frame and highlights detected faces with bounding boxes.

This project is ideal for beginners to understand basic computer vision concepts such as image processing, object detection, and real-time video handling.

---

## Features
- Real-time face detection using webcam
- Detects multiple faces in a frame
- Lightweight and fast processing
- Easy to understand and implement
- Extendable for face recognition projects

---

## Technologies Used
- Python
- OpenCV
